# CIE-11 Acción Socioeducativa Pro

Demo web profesional para orientar intervención socioeducativa a partir de un diagnóstico clínico confirmado con referencia CIE-11.

La herramienta incluye:

- selección de categorías CIE-11 de ejemplo validables en WHO;
- confirmación declarada de diagnóstico clínico;
- lectura socioeducativa del caso: convivencia, red, etapa vital y alertas sociales;
- urgencia social orientativa;
- recursos priorizados por municipio y necesidad;
- fuentes oficiales y bases de datos conectables;
- plan socioeducativo exportable desde el navegador.

Fuentes enlazadas:

- WHO ICD-11 Browser 2026-01: https://icd.who.int/browse/2026-01/mms/es
- WHO ICD API: https://icd.who.int/docs/icd-api/APIDoc-Version2/
- datos.gob.es API: https://datos.gob.es/es/apidata
- IMSERSO SAAD: https://imserso.es/autonomia-personal-dependencia/sistema-autonomia-atencion-dependencia-saad
- portales oficiales de Barcelona, Madrid, Sevilla, Comunidad de Madrid y Junta de Andalucía.

Límite ético: la web no diagnostica, no valida criterios clínicos y no sustituye atención sanitaria. Ordena necesidades, recursos y cautelas para intervención socioeducativa contextualizada.
